### Steam库存一键出售

[安装地址](https://greasyfork.org/zh-CN/scripts/35770-steam%E5%BA%93%E5%AD%98%E4%B8%80%E9%94%AE%E5%87%BA%E5%94%AE%E5%8A%9F%E8%83%BD)

库存和重复的卡太多的时候很实用

毕竟一个个写价格确认好几次太麻烦了

会自动获取起价，默认发售价格为起价+0.02

可安装后在脚本代码中自由调整
